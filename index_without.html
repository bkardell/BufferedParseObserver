Non-BufferedParseObserver Demo

This page is purely for use in collecting comparative data about the speed with which you can reach the DOMContentLoaded state without a BufferedParseObserver with various browsers, speeds, etc. It doesn't really demonstrate much of anything useful, it's value is in the fact that we can record timings and collect and compare them. For all intents and purposes this page aims to be as truthfully close as possible to the demo with the simple exception that it lacks the buffered parse observer. Since loading this incurs a network/block penalty in the demo (as a prollyfill), it should make for interesting, but somewhat hard to speculate on data.... At least it is something.
